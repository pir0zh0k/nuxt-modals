# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-XX

### Added
- Инициализация проекта
- Базовый функционал управления модальными окнами
- Поддержка TypeScript с полной типобезопасностью
- Композаблы `useModal` и `useInitModalConfig`
- Автоматический компонент `ModalContainer`
- Helper функции `createModalVariants` и `createModalConfig` для типизации
- Паттерн Singleton для инициализации конфигурации
- Поддержка вариантов оберток модальных окон (variants)
- Callback функции для открытия и закрытия модальных окон
- Полная документация с примерами использования
- Тесты для всех функций модуля
- SSR совместимость
